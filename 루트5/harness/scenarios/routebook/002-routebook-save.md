# Scenario 002: 루트 실행 후 루트북 저장

## 목표

사용자가 추천 루트를 선택하고 실행한 뒤, 사진과 글을 포함해 내 루트북에 저장한다.

## Given

- 사용자는 추천 후보 중 하나를 선택했다.
- 선택한 루트에는 5개 이하의 장소가 있다.
- 사용자는 루트 실행 화면에 있다.

## When

사용자가 모든 장소를 체크하고 `오늘의 루트 성공` 화면에서 사진, 제목, 글을 입력한 뒤 저장한다.

## Then

- 루트북 기록이 생성된다.
- 기록에는 날짜, 선택한 루트, 사진, 제목, 글이 포함된다.
- 저장 후 내 루트북 목록에서 해당 기록을 볼 수 있다.
- 사용자는 루트북 공유 또는 다른 루터에게 추천 액션을 선택할 수 있다.
