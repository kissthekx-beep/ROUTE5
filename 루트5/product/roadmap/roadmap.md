# 루트5 개발 로드맵

## 1단계: 정적 웹 프로토타입

목표: Netlify에서 바로 열 수 있는 링크를 만든다.

- 감정 선택
- 세부 조건 입력
- 고정 데이터 기반 루트 5개 추천
- 루트 선택
- 실행 체크리스트
- 루트북 저장 화면

## 2단계: 디자인 정리

목표: 프로토타입 드로잉의 화면 흐름을 앱 UI로 옮긴다.

- 상단 `내정보`
- 감정 아이콘
- 초록색 타이틀 박스
- 코랄색 액션 버튼
- 루트 노선형 체크리스트
- 루트북 카드와 목록

## 3단계: 장소 데이터 확장

목표: 서울 장소 100개를 직접 관리한다.

- 지역
- 카테고리
- 감정 태그
- 예상 비용
- 체류 시간
- 지도 검색 키워드

## 4단계: AI 연결

목표: 후보 장소 안에서 자연스러운 하루 흐름을 생성한다.

- 사용자 조건 해석
- 장소 후보 조합
- 루트 설명 생성
- 루터다운 말투 적용
- JSON 구조화 응답 검증

## 5단계: 앱 전환

목표: Expo / React Native로 아이폰에서 실행되는 앱을 만든다.

- 웹 프로토타입 화면 이식
- Expo Go 테스트
- 이후 TestFlight 준비

## 6단계: 실제 장소 API와 지도

목표: Google Places, 지도, 길찾기를 붙인다.

- 장소 영업 여부 확인
- 장소 상세 정보
- 네이버 지도 또는 Google Maps 연결
- 이동 시간 계산
- 지도 위 루트 표시
