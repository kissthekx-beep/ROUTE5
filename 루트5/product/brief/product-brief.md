# 루트5 제품 정리

## 한 줄 정의

루트5는 오늘의 기분, 예산, 이동수단에 맞춰 서울에서 실제로 실행 가능한 하루 루트 5곳을 추천하는 앱이다.

## 이름 정리

- `루트5`: 앱과 프로젝트 이름
- `루터`: 루트를 만들고 저장하고 공유하는 사용자 또는 추천자
- `슈퍼루터`: 검증된 큐레이션 루트를 제공하는 추천자
- `루트북`: 사용자가 실행하거나 저장한 루트 기록

## 핵심 가치

루트5는 검색 결과를 많이 보여주는 앱이 아니라, 오늘 하루를 대신 짜주는 앱이다. 사용자는 감정과 조건만 알려주고, 앱은 실제 장소 데이터 안에서 예산과 동선에 맞는 하루 계획을 만든다.

## 핵심 사용자

- 혼자 시간을 보내고 싶은 사람
- 아침에 오늘 뭘 할지 막막한 사람
- 데이트나 친구 약속 코스를 빠르게 짜고 싶은 사람
- 전시, 카페, 산책, 맛집을 좋아하지만 매번 검색하기 지친 사람

## MVP 범위

1. 감정 선택
2. 동행 인원, 이동수단, 거리, 예산, 취미 입력
3. 서울 장소 데이터 기반 루트 추천
4. 후보 루트 5개 탐색
5. 선택한 루트 실행 체크리스트
6. 루트 성공 후 사진, 제목, 글을 포함한 루트북 저장
7. 내 루트북과 다른 루터의 루트북 구경

## 추천 원칙

- AI는 장소를 상상하지 않는다.
- 장소, 가격, 지역, 카테고리, 감정 태그는 앱이 가진 데이터에서 고른다.
- AI는 후보 장소를 자연스러운 하루 흐름으로 묶고 말투를 정리한다.
- 처음에는 한 지역 안의 루트만 추천한다.
- 하루 루트는 최대 5개 장소로 구성한다.
- 추천 후보 루트도 최대 5개까지 보여준다.

## 초기 기술 방향

1. HTML, CSS, JavaScript로 정적 웹 프로토타입을 만든다.
2. Netlify Drop으로 친구나 지인에게 링크를 보내 피드백을 받는다.
3. 서울 장소 100개를 직접 데이터로 정리한다.
4. OpenAI API는 구조화된 추천 문구와 루트 설명에 붙인다.
5. 실제 앱 전환 시 Expo / React Native로 옮긴다.
6. 이후 Google Places API, 지도, 길찾기 기능을 붙인다.

## 대표 소개 문장

혼자 있는 하루도, 루트5가 하나의 루트로 만들어줍니다.
