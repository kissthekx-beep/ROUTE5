# routes

루트5 3페이지의 루트 후보 데이터입니다.

- 한 루트는 기본 5곳 이상으로 구성할 수 있습니다.
- 화면에는 한 번에 루트 1개를 보여주고, `다른루트 보기 1/5` 버튼으로 후보를 순환합니다.
- 각 후보에는 `tone`, `radiusKm`, `preferredTransports`, `moveMeters`, `tags`를 붙입니다.
- `tone`은 +적 기분/회복형 기분을 구분하는 추천 기준입니다.
- `moveMeters`는 가는방법에서 `120m 도보이동`처럼 보일 이동 거리입니다.
- `tags`에는 취미 단어와 장소 감각을 넣어서 `강아지`, `스포츠`, `음악`, `전시`, `책` 같은 입력이 실제 루트 순서에 반영되게 합니다.
- 현재 장소 사진은 카테고리별 임시 대표 이미지를 사용합니다.
- 정식 서비스에서는 슈퍼루터의 루트북 사진을 stop 데이터의 `photoSrc`로 연결합니다.
- 현재는 프로토타입용 정적 데이터입니다.
- 실제 서비스 단계에서는 네이버/구글 리뷰가 좋은 장소와 실제 위치/영업정보를 연결해야 합니다.
