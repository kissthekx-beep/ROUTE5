# apps/web

루트5의 웹 MVP입니다.

초기 목표는 HTML, CSS, JavaScript로 Netlify에 바로 배포 가능한 프로토타입을 만드는 것입니다.

## 현재 진입점

- `index.html`: 루트5 첫 화면
- `src/main/styles.css`: 첫 화면 스타일
- `src/main/app.js`: 감정 선택과 내정보 메뉴 상태 조립
- `src/features/mood-intake/moods.js`: 첫 페이지 감정 목록과 응답 문구

## 첫 페이지 방향

- 루트5는 혼자인 사람들을 위한 계획표입니다.
- 첫 페이지는 "오늘 너의 형태는?"을 묻고, 사용자의 기분을 수집합니다.
- 두 번째 페이지는 "좀 더 너의 형태를 알려줘!"를 묻고, 이동수단/거리/동행/비용/취미를 수집합니다.
- 디자인은 Studio Gaufrettes 테스트 페이지처럼 컬러풀한 캐릭터가 움직이는 테스트형 경험으로 통일합니다.
- BI 색상은 핑크 배경 `#df72c1`, 핑크 `#ff7bbe`, 블루 `#319aff`를 중심으로 합니다.
- 사용자가 기분을 클릭하면 선택된 칸은 검은색 바탕과 흰색 글자로 바뀝니다.
- 루티 캐릭터 이미지는 확정 전까지 웹 첫 화면에서 사용하지 않고, 선 없는 찰흙 질감의 컬러 덩어리 캐릭터를 사용합니다.
- 굵은 검은 외곽선은 사용하지 않습니다.

- `src/main`: 앱 부트스트랩, 라우팅, 전역 프로바이더
- `src/features`: 사용자에게 보이는 기능 단위
- `src/shared`: 앱 내부에서 공유하는 유틸리티와 컴포넌트
- `public`: 정적 파일
- `config`: 앱 환경 설정

이 폴더는 조립 계층으로 유지합니다. 도메인 규칙은 `packages/core`, 장소 데이터 조회는 `packages/catalog`, 추천 조합은 `packages/recommendation`, 외부 연동은 `packages/platform`에 둡니다.
