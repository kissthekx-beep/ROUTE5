# 루트5 프로토타입 화면 흐름

참고 이미지: `design/assets/root5-prototype-drawing.jpg`

## 색상 메모

- 파란색: 부연 설명과 화면 이동 안내
- 연두색: 클릭해도 이동하지 않는 안내성 문구
- 분홍색: 클릭하면 이동하는 액션 또는 텍스트

## 화면 1: 감정 선택

타이틀: `오늘 너의 형태는?`

상단에는 `내정보` 버튼이 있다. 사용자는 감정 아이콘 중 하나를 선택한다.

초기 감정:

- 그냥
- 행복
- 슬픔
- 설렘
- 우울

## 화면 2: 세부 조건 입력

타이틀: `조금더 자세히 알려줘!`

입력 항목:

- 혼자 / 둘 / 셋 / 넷 이상
- 도보 / 자동차 / 대중교통 / 기차
- 거리
- 비용
- 나의 취미

## 화면 3: 추천 방식 선택

앱은 사용자가 고른 조건을 문장으로 요약한다.

예시:

`슬픈 오늘, 미술을 좋아하고 혼자 있고 싶은 너에게 도보를 이용해서 1km 안에 10000원으로 할 수 있는 루트를 5가지 추천해줄게`

선택지:

- AI 추천
- 슈퍼루터 추천

## 화면 4: 슈퍼루터 추천 후보

헤더: `1/5 슈퍼루터 추천`

루트 후보는 시간순 목록으로 보여준다. 각 장소는 장소명, 한 줄 설명, 가는 길, 리뷰 링크를 가진다.

하단 액션:

- 선택
- 다른 루트

`다른 루트`를 누르면 다음 후보로 이동한다.

## 화면 5: AI 추천 후보

헤더: `1/5 ai 추천`

슈퍼루터 추천과 같은 레이아웃을 사용하되, 추천 생성 주체를 AI로 표시한다.

## 화면 6: 오늘의 루트 실행

타이틀: `오늘의 루트를 실행해보자`

선택한 루트를 체크리스트 또는 노선형 리스트로 보여준다. 사용자는 각 장소를 실행하면서 체크한다.

## 화면 7: 루트 성공 기록

타이틀: `오늘의 루트 성공!`

입력:

- 루트 정리하기
- 사진 올리기
- 루트 제목
- 글쓰기

액션:

- 내 루트북에 저장
- 다른 루터에게 추천

## 화면 8: 내정보 메뉴

`내정보`를 클릭하면 메뉴가 열린다.

메뉴:

- 내루트북
- 다른 루터북 구경
- 슈퍼루터
- 오늘의 루트 선택하기

## 화면 9: 내 루트북

헤더: `루트북`

날짜별 루트 기록을 보여준다.

하단 액션:

- 루트북 공유
- 루터북 구경

## 화면 10: 다른 루터북 구경

헤더: `루터북`

다른 루터의 루트북 목록을 보여준다.

예시 카테고리:

- 감성적인
- 활동적인
